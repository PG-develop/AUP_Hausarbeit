\doxysubsection{Ausführliche Beschreibung}
Zahlenratespiel. 

Es wird eine Zahl generiert die ein Benutzer herausfinden muss.

Um die Option mit begrenzter Anzahl an Versuchen zu aktiveren, m\+Max\+Tries mit einer anderen Zahl als 0 initialisieren.
