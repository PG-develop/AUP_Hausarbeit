Wechsel zu einem anderen \mbox{\hyperlink{class_a_u_p___h_a_1_1_state}{State}}. 

Wechselt auf den neuen \mbox{\hyperlink{class_a_u_p___h_a_1_1_state}{State}}, der mit dem angegeben Zustandsidentifiziierer registriert wurde. Der gewünschte \mbox{\hyperlink{class_a_u_p___h_a_1_1_state}{State}} muss vorher registriert worden sein. Ansonsten kommt es zu einem Programmfehler.

Der aktive \mbox{\hyperlink{class_a_u_p___h_a_1_1_state}{State}} wird gelöscht und der geforderte \mbox{\hyperlink{class_a_u_p___h_a_1_1_state}{State}} wird durch die Zustandsfabrik instanziiert. Wenn das Programm beendet werden soll muss der Zustandsidentifiziierer \mbox{\hyperlink{namespace_a_u_p___h_a_1_1_states_aba7518c4c9fcad80b303d82603931711a3618a329870f8c662a34cd9b4045cacc}{States\+::\+NONE}} ausgewählt werden.
