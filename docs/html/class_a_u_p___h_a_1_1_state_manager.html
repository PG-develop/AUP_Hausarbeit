◆ changeState()

void AUP_HA::StateManager::changeState ( States::ID  stateID)

Wechsel zu einem anderen State.

Wechselt auf den neuen State, der mit dem angegeben Zustandsidentifiziierer registriert wurde. Der gewünschte State muss vorher registriert worden sein. Ansonsten kommt es zu einem Programmfehler.

Wenn der gewünschte State bereits aktiv ist, wird dieser nicht erneut instanziiert.

Parameter
[stateID]States::ID Zustandsidentifiziierer

template<typename T >
void AUP_HA::StateManager::registerStates ( States::ID  stateID)
inline

Registriert der Zustände in der Zustandsfabrik.

Mithilfe eines Identifizieres werden die Zustände registriert. Es wird jedoch keine Instanz erstellt. Die Registrierten Zustände können mithilfe der Methode "changeState" instantiiert werden. Mithilfe des Template-Parameters wird festgelegt, welcher State instanziiert werden soll. Die Klasse muss von State erben.
