Übergangsbedingnung NH.

Prüft, ob die Eingabe die gesuchte Zahl ist
Wenn die gesuchte Zahl gefunden wurde, dann wechsel zum GameState STATE_5
Wenn die gesuchte Zahl nicht gefunden wurde, dann wechsel zum GameState STATE_2

Bei einer beschränkten Versuchsanzahl wird geprüft, ob die maximale Anzahl an Rateversuchen erreicht ist. Wenn diese erreicht wurde, wechsle zum GameState STATE_6

template<typename T >
void AUP_HA::NumberGameStateManager::RegisterStates ( GameStates::ID  stateID_p)
inline

Registriert der Zustände in der Zustandsfabrik.

Mithilfe eines Identifikators werden die Zustände registriert. Es wird jedoch keine Instanz erstellt. Die Registrierten Zustände können mithilfe der Methode "ChangeState" instantiiert werden. Mithilfe des Template-Parameters wird festgelegt, welcher State instanziiert werden soll. Die Klasse muss von State erben.

Template-Parameter
[T]State Geerbte Klasse von NumberGameState
Parameter
[stateID_p]GameStates::ID Zustandsidentifikator
